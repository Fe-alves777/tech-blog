Tags de texto

Voltar inicio

Essas são umas das tags mais comuns e utilizadas durante o desenvolvimento da sua aplicação HTML, usamos essas tags para escrevermos no HTML de forma ordenada, e cada uma das tags trabalham de forma distinta para titulos por exemplo usamos < h1 > e temos variações de titulos que vão até < h6 >, tambem temos a tag < p > que é uma forma de escrevermos um parágrafo, temos o <blockquote > que serve para nós escrevermos citações nos nossos textos além de outras mais


Tag de imagem

Voltar inicio

A tag de imagem <img> é utilizada para adicionar pasmem... imagens ao nosso HTML essa tag é composta por uma propriedade obrigatória que é o src='caminho__da__imagem' essa tag pode receber tanto imagens que estão nos arquivos da aplicação como links externos, e a outra propriedade que embora não seja obrigatória, é bastante recomendada que é a alt='descrição__imagem' essa é uma propriedade que traz responsavidade a nossa página com uma descrição para a imagem, que aparece quando a imagem não pode ser carregada e também é "falada" por sistemas responsivos para cegos, além disso ajuda no SEO do site.


Tag de link

Voltar inicio

Essa tag <a> é chamada de tag anchor ou âncora, ela nos permite ancorar links para páginas externas, seções da nossa página ou até mesmo outras páginas ainda dentro do nosso site ela permite isso ao anexar hyperLinks em textos, esses links podem ser abertos em uma nova aba ou na atual esses detalhes são passados para a tag pela propriedade target='', quando o link for para um site externo é recomendado que abramos em uma nova aba, se for para o nosso site devemos manter na mesma aba e apenas atualizar o conteúdo, essas são as boas práticas.
